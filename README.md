# Game Hub — Браузерные игры для двоих

Платформа: заходите на сайт, вводите имена, выбираете игру и играете вдвоем.

## Запуск

```bash
npm install && npm start
```

Откройте `http://localhost:3000` на двух устройствах.

## Игры

- **Тетрис** — Battle и Co-op режимы
- **Pong** — классический понг, счёт до 11
- **Крестики-нолики** — классика на двоих
- **Четыре в ряд** — стратегия
- **Пары** — найди все пары карточек

## Управление

| Игра | Desktop | Mobile |
|------|---------|--------|
| Тетрис | ← → ↑ ↓ Space | D-пад |
| Pong | ↑ ↓ / W S | Кнопки |
| Остальные | Клик/тап | Тап |

## Стек

Node.js + WebSocket (`ws`) + Vanilla JS + Canvas + CSS
